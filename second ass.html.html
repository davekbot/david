An internet is a computer networking system used to connect to different users.
   A browser is a software that renders webpages.
  Characteristics in order to view a website
  1. browsers
  2. internet connectivity
  3. A server is  a  dedicated machine used to serve out Page's to client 24/7
    Url: uniform resource locator
    How does the website work?
    1. Firstly the client sends url to to the isp(internet service provider)
    2. Isp sends it to a dns server(domain name system)
    N;b ip address is the protocol the internet uses for communication.
    The protocols for websites are "http" ; hyper text transfer protocol
    The first page of every website is always the "homepage"

    The homepage consists of ;
    1. Html
    2. Css
    3. JavaScript
    Clients are machines that have internet connectivity and have a working browser also.
